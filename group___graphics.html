int guLoadTexture ( char *  filename,
GLuint *  texture 
)

Loads a texture from game memory (Assets), not from disk.

Parameters
filenamefile name of the texture file just as it appears in the "textures" folder, for example "wall.png".
[out]texturethe texture address.

The dimensions of a texture must be power of 2, the maximum allowed is 512x512. It is recommended to use dimensions less than or equal to 256x256, and the width does not need to be equal to the height.

GLuint tex_box;
guLoadTexture("box.png", &tex_box);

Loads a 3D model from game memory (Assets), not from disk.

Parameters
filenamefile name of the SMD file just as it appears in the "models" folder, for example "box.smd".
bakeableGU_BAKEABLE, GU_NOBAKEABLE.
[out]modelPointer to model.
GUModel model_box;
guLoadModel("box.smd", GU_NOBAKEABLE, &model_box);
Loading a model with GU_BAKEABLE
Implies that the colors of its vertices (which may be related to shading) will be captured in your first drawing in order to use them in the following drawings without making shading calculations. This is optimal when the model is static as well as the lights that affect it.
Loading a model with GU_NOBAKEABLE
implies that shading calculations will always be performed on each drawing.
Blender for export model in SMD format
I recommend using blender to create and export models in SMD format. In blender when exporting a model in SMD format, make sure the "Target Up Axis" is set to "Y", so: X(gu3d)=X(blender), Y(gu3d)=Z(blender), Z(gu3d)=-Y(blender).
